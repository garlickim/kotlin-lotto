# 문자열 계산
[구현 기능 목록]
- 커스텀 구분자 포함 여부 판단
- 커스텀 구분자 파싱
- 구분자로 파싱
- 입력된 값이 자연수인지 확인 ( >= 0 )
- 숫자들의 합계


# 로또 (자동)
- 구입 금액 입력 (default = 0)
- 로또 구매 개수 구현
- 구매 개수 만큼 로또 생성
- 로또 노출
- 지난주 당첨번호 입력
- 당첨 통계 계산
- 수익률 계산
- 통계 노출

# 로또 (2)
- 2등 추첨을 위해 보너스 볼 입력
- enum 클래스에 보너스 추가
- 보너스 번호 포함 5자리 당첨 됐는지 판단
- 수익률 계산에 보너스 당첨 추가
